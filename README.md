# CoaL
**Co**de **a**nd **L**earn

**Free** html editor + 20 scripts for learning

* Write or paste html code
* click **Run** or press **Shift Enter**

*the science is in, and it's CoaL!*

**Document**: `Elements`, `Style`, `Identifiers` `Containers`, `Layout`
